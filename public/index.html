Jeopardy! Game Master

Game setup is as easy as 1, 2, 3, (4)!

Step 1 (Easy): Game creation using default games

The Jeopardy Game Master database is initially empty. While there are options below that allow you to create a Jeopardy game completely from scratch (see Step 1 (Alternate)), the easiest way to start playing is to load the default Jeopardy and Double Jeopardy games into the database. You can do this by selecting the Transfer Defaults button.

Step 1 (Alternate): Create your own categories and/or game!

A Jeopardy game layout includes six categories which are presented as columns on the game board. Each category holds five questions connected via a common theme. To create a custom category select the Create Category button. To create a custom game select the Create Game button.

Step 2

After you have edited your Jeopardy games, it's time to choose one game to present to the players. To choose a game select the Choose Game button.

Step 3

To play Jeopardy select the Play Game button.

Step 4 (Optional): Run Route Tests

The Jeopardy application is designed with a "virtual database manager" that sits in between the game authoring/playing screens and the database. The route tests bypass the user interface portions of the application and directly excercise the six routes that server.js supports. These route tests can be seen in the file vdbman.js in function runRouteTests. To run the route unit tests select the Run Route Tests button.
